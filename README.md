# ঔষধ মনে রাখুন - Medicine Reminder App

একটি বাংলা ভাষায় তৈরি Progressive Web App (PWA) যা আপনার ঔষধ খাওয়ার সময় মনে রাখতে সাহায্য করবে।

## ফিচারসমূহ

- ✅ বাংলা ইন্টারফেস
- ✅ ঔষধ যোগ, সম্পাদনা ও ডিলিট
- ✅ টাইমার সিস্টেম
- ✅ ডেটা ব্যাকআপ ও রিস্টোর
- ✅ অফলাইন কাজ করে
- ✅ ফোনে ইন্সটল করা যায়

## ব্যবহার বিধি

1. অ্যাপটি ওপেন করুন
2. "নতুন ঔষধ যোগ করুন" ক্লিক করুন
3. ঔষধের তথ্য দিন
4. ঔষধ খেয়ে "খেয়েছি?" বাটনে ক্লিক করুন

## GitHub Pages-এ ডেপ্লয় করা

সহজেই GitHub Pages-এ ডেপ্লয় করতে:

1. এই রিপোজিটরি ফর্ক করুন
2. Settings > Pages-এ গিয়ে Source সিলেক্ট করুন
3. Save করুন
